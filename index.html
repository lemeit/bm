⚖️ Balances de Materia en Ingeniería de Procesos

Fundamentos, metodología sistemática y verificación con Python.
Basado en Felder & Rousseau (2004) y Himmelblau (1997) – Procesos no reactivos en estado estacionario.

📚 Contexto y principios esenciales (Felder, Himmelblau):
• Ley de conservación de la masa: “La materia no se crea ni se destruye”. En procesos no reactivos sin acumulación: Entrada = Salida.
• Sistemas y fronteras: Un sistema puede ser abierto (transferencia de masa) o cerrado. Para análisis se define una frontera (volumen de control).
• Procesos en estado estacionario: Las variables (flujos, composiciones, temperatura) no cambian en el tiempo ⇒ Acumulación = 0.
• Balance total y por componentes: Se escribe un balance para cada especie química presente. Suma de fracciones másicas o molares = 1.
• Grados de libertad: Número de variables desconocidas menos ecuaciones independientes. Para resolver un problema se requiere G.L. = 0.
• Metodología propuesta por Felder: 1) Leer/diagramar, 2) Etiquetar corrientes, 3) Elegir base de cálculo, 4) Escribir balances, 5) Resolver, 6) Verificar.
• Importancia en ingeniería: Los balances de materia son la base del diseño, simulación y optimización de procesos químicos y biotecnológicos.

📐 Procedimiento sistemático para resolver balances (no reactivos)

✅ Paso a paso (adaptado de Felder & Rousseau, Cap. 4):
1. Interpretar el problema: identificar datos explícitos e implícitos, y las incógnitas.
2. Dibujar diagrama de flujo: representar equipos, corrientes y composiciones conocidas. Usar símbolos para incógnitas.
3. Elegir base de cálculo: tiempo (ej. 1 hora) o masa (ej. 100 kg), que facilite los cálculos.
4. Analizar grados de libertad: comparar número de incógnitas vs. ecuaciones independientes (balances más relaciones auxiliares).
5. Plantear ecuaciones: balance global (total mass in = total mass out) y balances por componente.
6. Resolver el sistema: algebraico, sustitución o métodos matriciales.
7. Verificar usando un balance no utilizado o chequeo de consistencia (fracciones suman 1).
💡 Nota: Los ejemplos siguientes aplican esta metodología; luego se incluye código Python para comprobación rápida.

📌 Ejemplo 3.1 – Mezcla de N₂ e H₂ (Felder tipo)

Enunciado: Una corriente de 280 kg/h de N₂ se mezcla con H₂ puro. La corriente de salida total es 40 kgmol/h. Calcular el flujo de H₂ en kgmol/h y la composición molar de la mezcla final.

✍️ Solución manual paso a paso:
• Base: 1 hora. Convertir N₂ a kgmol: (280 kg) / (28 kg/kgmol) = 10 kgmol/h.
• Balance global: N₂ + H₂ = Total salida → 10 + B = 40 ⇒ B = 30 kgmol/h de H₂.
• Balance por componente (H₂): B·1 = 40·yC,H₂ ⇒ yC,H₂ = 30/40 = 0.75, yC,N₂ = 0.25.
• Verificación N₂: 10 = 40·0.25 ✔️.
🐍 Verificación con Python (cálculo numérico):
# Ejemplo 3.1 - Mezclador simple flujo_N2_kg = 280 M_N2 = 28 flujo_N2_kmol = flujo_N2_kg / M_N2 # 10 kmol/h C_total_kmol = 40 # kmol/h salida B_kmol = C_total_kmol - flujo_N2_kmol # H2 yH2 = B_kmol / C_total_kmol yN2 = 1 - yH2 print(f"H₂ en entrada: {B_kmol} kgmol/h") print(f"Fracción molar H₂ = {yH2:.3f}, N₂ = {yN2:.3f}") # Resultados: 30 kgmol/h, yH2=0.750, yN2=0.250

📌 Ejemplo 3.2 – Destilación con tres componentes (Himmelblau)

Datos: F=32 kg/h (12.5% B, 87.5% C), T=28 kg/h (100% A), M=35 kg/h (28.6% B), desconocidas E, D, composición de M (% A). Hallar E, D, xM,A.

✍️ Metodología:
• Balance global: 32 + E = 28 + 35 + D → E - D = 31 (1)
• Balance componente B: 32·0.125 + E·0.466 = 35·0.286 + D·0.614 → 0.466E - 0.614D = 6.01 (2)
• Resolver: de (1) E = D+31; sustituir: 0.466(D+31) - 0.614D = 6.01 → D = 57 kg, E = 88 kg.
• Balance A: 88·0.534 = 28·1 + 35·xM,A → xM,A = 0.543, luego xM,C = 0.171.
🐍 Solución con NumPy:
import numpy as np # Sistema para E y D: E - D = 31 ; 0.466E - 0.614D = 6.01 A_mat = np.array([[1, -1], [0.466, -0.614]]) b_vec = np.array([31, 6.01]) E, D = np.linalg.solve(A_mat, b_vec) x_MA = (E*0.534 - 28) / 35 x_MC = 1 - 0.286 - x_MA print(f"E = {E:.1f} kg/h, D = {D:.1f} kg/h") print(f"x_MA = {x_MA:.3f}, x_MC = {x_MC:.3f}")

📌 Ejemplo 3.3 – Secado (pérdida de humedad)

Planteamiento: Piel salida = 900 lb con 7% humedad → piel seca = 837 lb. Durante secado se perdió 59.1% del peso inicial. Calcular A (alimentación húmeda), agua extraída, % eliminación.

✍️ Solución:
• Piel seca: A·(1 - xA,agua) = 837. Balance agua: A·xA,agua = 0.591A + 900·0.07 = 0.591A + 63.
• Además A·xA,agua = A - 837 → A - 837 = 0.591A + 63 → 0.409A = 900 → A = 2200 lb.
• Agua eliminada B = 0.591*2200 = 1300 lb; agua inicial = 1363 lb; % eliminado = 95.6%.
• Agua por lb piel seca = 1300/837 = 1.553.
🐍 Código de verificación:
C = 900 xC_agua = 0.07 piel_seca = C * (1 - xC_agua) # 837 lb # A - 0.591A = piel_seca + C*xC_agua A = (piel_seca + C*xC_agua) / (1 - 0.591) B = 0.591 * A agua_inicial = A - piel_seca porc_elim = B / agua_inicial * 100 agua_por_lb_seca = B / piel_seca print(f"Alimentación A = {A:.1f} lb, Agua extraída B = {B:.1f} lb") print(f"% agua eliminada = {porc_elim:.1f}%, lb agua/lb piel seca = {agua_por_lb_seca:.3f}")

📌 Ejemplo 3.4 – Dos unidades de separación (balances secuenciales)

Descripción: Corrientes E=100 kg (50% A), F=40 kg (90% A), H=30 kg (30% A), N=30 kg (60% A). Encontrar G, M, P, xG,A, xP,A, xM,A.

✍️ Solución escalonada:
• Unidad 1: 100 = 40 + G → G = 60 kg; Balance A: 100·0.5 = 40·0.9 + 60·xG,A → xG,A = 0.2333.
• Global sistema: 100+30 = 40+30+P → P = 60 kg; Balance A global: 100·0.5+30·0.3 = 40·0.9+30·0.6+60·xP,A → xP,A=0.08333.
• Punto mezcla M: M = N+P = 90 kg; xM,A = (30·0.6+60·0.08333)/90 = 0.2556.
🐍 Python secuencial:
E, xE_A = 100, 0.5 F, xF_A = 40, 0.9 H, xH_A = 30, 0.3 N, xN_A = 30, 0.6 G = E - F xG_A = (E*xE_A - F*xF_A)/G P = E + H - F - N xP_A = (E*xE_A + H*xH_A - F*xF_A - N*xN_A)/P M = N + P xM_A = (N*xN_A + P*xP_A)/M print(f"G={G} kg, xG_A={xG_A:.4f}; P={P} kg, xP_A={xP_A:.5f}") print(f"M={M} kg, xM_A={xM_A:.4f}")

📌 Ejemplo 3.5 – Separación fraccionada con 3 componentes

Condiciones: F=855 kg/h (20% X, 40% B, 40% T). D con 90% X, A con 99% B, C con 5% B. Determinar A, C, D, B y composición de B.

✍️ Resolución:
• Balance Xileno global: 855·0.2 = D·0.90 → D = 190 kg.
• Balance Benceno global: 342 = 0.99A + 0.05C, y global total: A + C = 855-190 = 665 → A=328.46, C=336.54.
• Unidad 1: B = F - A = 527 kg; xB,X = (855*0.2)/527 = 0.324; xB,B = (342-328.46*0.99)/527 = 0.0328; xB,T = 1 - 0.324 - 0.0328 = 0.643.
🐍 Con SymPy para mayor claridad:
from sympy import symbols, solve F, xF_X, xF_B = 855, 0.2, 0.4 xD_X = 0.9; xA_B = 0.99; xC_B = 0.05 D = (F * xF_X) / xD_X # 190 A, C = symbols('A C') eq1 = A + C - (F - D) eq2 = 0.99*A + 0.05*C - F*xF_B sol = solve((eq1, eq2), (A, C)) A_val = float(sol[A]); C_val = float(sol[C]) B = F - A_val xBX = (F*xF_X) / B xBB = (F*xF_B - A_val*xA_B) / B xBT = 1 - xBX - xBB print(f"A={A_val:.1f} kg, C={C_val:.1f} kg, D={D:.1f} kg, B={B:.1f} kg") print(f"x_B,X={xBX:.3f}, x_B,B={xBB:.4f}, x_B,T={xBT:.3f}")

📌 Ejemplo 3.6 – Proceso con recirculación (ejemplo clásico Himmelblau)

Datos: F=10000 lb/h, 20% KNO₃, alimentación a evaporador. R: disolución saturada (0.6 lb KNO₃/lb H₂O → xR=0.375). Cristales C con 96% KNO₃. Corriente M 50% KNO₃. Calcular R, M, W, E y composición de E.

✍️ Estrategia:
• Balance global de KNO₃: 10000·0.2 = C·0.96 → C = 2083.33 lb; luego W = F - C = 7916.67 lb.
• Cristalizador: M = R + C; Balance KNO₃: 0.5M = 0.96C + 0.375R → 0.5(R+C)=0.96C+0.375R → R = 7666.67 lb, M = 9750 lb.
• Mezcla E = F + R = 17666.67 lb; xE,H2O = (10000·0.8 + 7666.67·0.625)/E = 0.724, xE,KNO3=0.276.
🐍 Código de verificación:
F = 10000 xF_KNO3 = 0.2 xC_KNO3 = 0.96 xR_KNO3 = 0.375 xM_KNO3 = 0.5 C = (F * xF_KNO3) / xC_KNO3 W = F - C # Cristalizador: 0.5(R+C) = 0.96*C + 0.375*R R = (0.96*C - 0.5*C) / (0.5 - 0.375) M = R + C E = F + R xE_H2O = (F*0.8 + R*0.625) / E print(f"C={C:.2f} lb/h, W={W:.2f} lb/h, R={R:.2f} lb/h, M={M:.2f} lb/h") print(f"E={E:.2f} lb/h, xE_H2O={xE_H2O:.4f}, xE_KNO3={1-xE_H2O:.4f}")

📌 Ejercicio de práctica – Mezcla Etanol/Agua (Felder & Rousseau 4.2)

Planteamiento: Corriente A (30% etanol) + Corriente B (60% etanol) producen C al 35% etanol. a) Determinar relación A/B. b) Si C=4500 kg/h, calcular A y B.

✍️ Solución analítica:
• Balance etanol: 0.3A + 0.6B = 0.35(A+B) → 0.3A+0.6B = 0.35A+0.35B → 0.25B = 0.05A → A/B = 5.
• Relación independiente del caudal. Para C=4500: A = (5/6)*4500 = 3750 kg/h, B = 750 kg/h.
🐍 Cálculo Python:
relacion_AB = 5 C_masa = 4500 A = (relacion_AB / (1 + relacion_AB)) * C_masa B = C_masa - A print(f"Relación A/B = {relacion_AB}") print(f"Para C = {C_masa} kg/h -> A = {A:.1f} kg/h, B = {B:.1f} kg/h") # Comprobación: 0.3*3750+0.6*750 = 1575 kg etanol, C*0.35=1575 ✔️

📊 Resumen de ecuaciones clave (procesos no reactivos, E.E.)

ConceptoEcuaciónAplicación típica
Balance globalΣ m_entrada = Σ m_salidaSistema completo o unidad
Balance por componenteΣ (m_i · x_i,ent) = Σ (m_j · x_j,sal)Cada especie química
Fracción másica/molarΣ x_k = 1Verificación de consistencia
Grados de libertadGL = N° incógnitas - N° ec. independientesDeterminar si el problema está bien especificado
