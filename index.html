📚 Fundamentos de Balance de Materia (Felder, Himmelblau):
• Ley de conservación de la masa: “La materia no se crea ni se destruye”. En procesos no reactivos sin acumulación: Entrada = Salida.
• Sistemas y fronteras: Un sistema puede ser abierto (transferencia de masa) o cerrado. Para análisis se define una frontera (volumen de control).
• Procesos en estado estacionario: Las variables no cambian en el tiempo ⇒ Acumulación = 0.
• Balance total y por componentes: Se escribe un balance para cada especie química presente. Suma de fracciones másicas o molares = 1.
• Grados de libertad: Número de variables desconocidas menos ecuaciones independientes. Para resolver se requiere G.L. = 0.
• Metodología propuesta por Felder: 1) Leer/diagramar, 2) Etiquetar corrientes, 3) Elegir base de cálculo, 4) Escribir balances, 5) Resolver, 6) Verificar.

📐 Procedimiento sistemático para resolver balances (no reactivos)

✅ Paso a paso (adaptado de Felder & Rousseau, Cap. 4):
1. Interpretar el problema: identificar datos explícitos e implícitos, y las incógnitas.
2. Dibujar diagrama de flujo: representar equipos, corrientes y composiciones conocidas.
3. Elegir base de cálculo: tiempo (ej. 1 hora) o masa (ej. 100 kg).
4. Analizar grados de libertad: comparar número de incógnitas vs. ecuaciones independientes.
5. Plantear ecuaciones: balance global y balances por componente.
6. Resolver el sistema: algebraico, sustitución o métodos matriciales.
7. Verificar usando un balance no utilizado.

📌 Ejemplo 3.1 – Mezcla de N₂ e H₂

Enunciado: Corriente de 280 kg/h de N₂ se mezcla con H₂ puro. Salida total: 40 kgmol/h. Hallar flujo de H₂ y composición molar.

✍️ Solución manual:
• Base 1h. N₂: 280/28 = 10 kgmol/h.
• Balance global: 10 + B = 40 → B = 30 kgmol/h H₂.
• yC,H₂ = 30/40 = 0.75; yC,N₂ = 0.25.
🐍 Verificación Python:
flujo_N2_kg = 280 flujo_N2_kmol = 280 / 28 C_total = 40 B = C_total - flujo_N2_kmol yH2 = B / C_total print(f"H₂: {B} kgmol/h, yH₂={yH2:.3f}")

📌 Ejemplo 3.2 – Columna de destilación

Datos: F=32 kg/h (xF,B=0.125), T=28 kg/h (100% A), M=35 kg/h (xM,B=0.286). Hallar E, D, xM,A.

✍️ Solución:
• Balance global: 32+E = 28+35+D → E-D=31.
• Balance B: 4 + 0.466E = 10.01 + 0.614D → E=88 kg/h, D=57 kg/h.
• Balance A: 88·0.534 = 28 + 35·xM,A → xM,A=0.543.
🐍 Python con NumPy:
import numpy as np A = np.array([[1, -1], [0.466, -0.614]]) b = np.array([31, 6.01]) E, D = np.linalg.solve(A, b) print(f"E={E:.1f}, D={D:.1f}")

📌 Ejemplo 3.3 – Secado de pieles

Planteamiento: Salida 900 lb con 7% humedad. Pérdida inicial 59.1%. Hallar alimentación, agua extraída y % eliminado.
