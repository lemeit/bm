# ⚖️ Balances de Materia - Simulador y Guía Interactiva

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📚 Contenido

- **Guía teórica** (`index.html`): Fundamentos de balances de materia según Felder & Rousseau y Himmelblau. Incluye ejemplos resueltos paso a paso y código Python de verificación.
- **Simulador interactivo** (`simu_destilacion.html`): Columna de destilación donde puedes cambiar caudales y composiciones, y ver cómo se reajustan automáticamente las incógnitas mediante balances de materia.

## 🎯 Objetivo didáctico

Que los estudiantes puedan:
1. Aprender la metodología sistemática para resolver balances.
2. Verificar sus cálculos manuales con Python.
3. **Experimentar visualmente** cómo afecta cambiar una variable en un proceso de separación.

## 🧪 Cómo usar el simulador

- Abre `simu_destilacion.html` en tu navegador.
- Modifica los valores de entrada (F, T, M, composiciones).
- El sistema resuelve automáticamente las corrientes desconocidas (E, D) y las composiciones de M.
- Observa las advertencias si los valores no son factibles.

## 🔧 Tecnologías

- HTML5, CSS3, JavaScript (Vanilla) – sin dependencias externas.
- Python (códigos de verificación incluidos en la guía).

## 📖 Referencias

- Felder, R. M. & Rousseau, R. W. (2004). *Principios elementales de los procesos químicos*.
- Himmelblau, D. M. (1997). *Principios básicos y cálculos en ingeniería química*.
